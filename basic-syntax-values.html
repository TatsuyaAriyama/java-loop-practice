Operator Lesson

演算子・型変換・スコープを、試験で崩れない読み方にする

Lesson01でclass、main、表示、変数を確認したら、次は値の種類と計算です。値を「箱に入れるもの」、型を「箱の種類」、演算子を「値を変える道具」として整理します。

class Main {
  public static void main(String[] args) {
    final int TAX_RATE = 10;
    int price = 500;
    int count = 2;
    int total = price * count;

    double average = 5.0 / 2;
    char grade = 'A';

    System.out.println("total: " + total);
    System.out.println(average);
    System.out.println(grade);
  }
}
type

型は、入れてよい値を決める

int は整数、double は小数、char は1文字です。箱の種類が違う、と考えると読みやすくなります。

=

= は、右を左の箱へ入れる

int total = price * count; は、まず右側を計算して、その答えをtotalへ入れます。

+

+ は、足し算にも文字つなぎにもなる

数字同士なら足し算、文字列が混ざると文字をつなげます。同じ+でも、まわりの値によって働きが変わります。

boolean

比較の答えはtrueかfalse

score >= 60 の答えは数ではなく、trueかfalseです。if文はこの答えを見て動きます。
