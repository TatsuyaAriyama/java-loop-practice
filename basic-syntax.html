Basic Syntax Lesson

本当にゼロから始めるJavaの最初の文法

Javaは、最初はいきなり長い英語のように見えます。でも分けると、classという大きな箱、mainという入口、画面に出す命令、値を入れる変数からできています。まずはここだけをゆっくり確認します。

class Main {
  public static void main(String[] args) {
    String name = "Aoi";
    int score = 80;
    boolean passed = score >= 60;

    System.out.println(name + ": " + score);

    Student student = new Student();
    student.name = name;
  }
}

class Student {
  String name;
}
class

Javaはclassの中にコードを書く

class Main は、Javaコードを入れる外枠です。処理は何もない場所に浮かず、基本的にクラスの中に置かれます。

main

mainは実行の入口

public static void main(String[] args) の中に書いた命令が、上から順番に実行されます。

variable

変数は値に名前を付ける

int score = 80; は、scoreという名前で80を扱えるようにする書き方です。

type

型は値の種類を決める

int、String、boolean の違いを読めると、コードの意味が崩れにくくなります。

=

= は右の値を左へ入れる

count = count + 1; は、右側を先に計算して、その結果をcountへ入れる処理です。

String

文字列と値は+でつなげる

"score: " + score のように、文字列と変数を連結して表示用の文字を作れます。

println

printlnは値を表示する

System.out.println(...) は、かっこの中の値を画面へ出します。出力問題ではここを丁寧に読みます。

scope

変数には使える範囲がある

波かっこの範囲を意識すると、ローカル変数、引数、フィールドの違いを見分けやすくなります。
