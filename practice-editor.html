+= を実際に動かしてみよう

このLessonは無料で使えます。コードの数字を書き換えて、実行結果がどう変わるかをその場で確かめます。

point 01

+= は「足してから代入」

x += 1 は、今のxに1を足して、もう一度xへ入れる書き方です。

point 02

x = x + 1 と同じ意味

短く書いているだけで、値が変わる流れは同じです。右側を先に計算します。

point 03

数字を変えると出力も変わる

x += 5; にすると、最終的に表示されるxも5になります。
