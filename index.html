for

回数や順番を決めて繰り返すときに使います。Javaでは、丸かっこの中を(1)初期化式、(2)条件式、(3)更新式の3つに分けて書きます。

for (初期化式; 条件式; 更新式)
  • 初期化式：最初の値を決める
  • 条件式：続けるか判定する
  • 更新式：1回ごとに値を変える

while

条件式がtrueの間だけ、同じ処理を繰り返します。回数よりも「この条件を満たしている間」という考え方が中心です。

while (条件式)
  • 条件式を先に確認する
  • trueなら中の処理を実行する
  • falseになったら繰り返しを終える

break

繰り返しをその場で終了します。たとえば「探している値が見つかったら、もう続けなくてよい」という場面で使います。

break;
  • 一番内側の繰り返しを抜ける
  • breakより後の処理は実行しない
  • 終了後は繰り返しの次へ進む

continue

繰り返し自体は続けたまま、今の1回分だけ残りの処理を飛ばします。「条件に合わない回だけスキップする」と考えるとわかりやすいです。

continue;
  • 今の回だけを途中で終える
  • 次の回の判定へ進む
  • 除外したい値があるときに便利

参考: Oracle Java Tutorials の for文、 while文、 break / continue
