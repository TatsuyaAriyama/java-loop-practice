Java Syntax Guide

コードの外枠を理解する

問題で何度も出てくる形を、知識ゼロでも読めるように分解します。

class Main {
  public static void main(String[] args) {
    // ここに処理を書く
    
  }
}

ヒント: Hello と表示する1行を書きます。

class Main

クラスは、Javaのコードを入れる大きな箱

Javaでは、処理をいきなり外に書かず、基本的にclassの中へ書きます。ここでは練習用の箱の名前をMainにしています。

main method

mainメソッドは、プログラムのスタート地点

public static void main(String[] args)は、Javaアプリを実行するときに最初に呼ばれる特別な形です。今は「ここから始まる合図」と覚えれば大丈夫です。

{ }

波かっこは、範囲を表す

{から}までが、ひとまとまりの範囲です。classの波かっこの中にmainメソッドがあり、mainメソッドの波かっこの中に実際の処理を書きます。

;

セミコロンは、文の終わりの印

System.out.println(i);のように、1つの命令の終わりには;を付けます。日本語の「。」に近い役割です。

// comment

コメントは、実行されないメモ

//から右側はコメントです。人間が読むためのメモで、Javaは実行するときに無視します。

println

printlnは、画面に表示して改行する命令

System.out.println(...)は、かっこの中の値を画面に出して、最後に改行します。問題の「目標の出力」は、この命令で作ります。

Before Coding

最初にここだけ押さえる

for

回数や順番を決めて繰り返すときに使います。Javaでは、丸かっこの中を(1)初期化式、(2)条件式、(3)更新式の3つに分けて書きます。

for (初期化式; 条件式; 更新式)
  • 初期化式：最初の値を決める
  • 条件式：続けるか判定する
  • 更新式：1回ごとに値を変える

while

条件式がtrueの間だけ、同じ処理を繰り返します。回数よりも「この条件を満たしている間」という考え方が中心です。

while (条件式)
  • 条件式を先に確認する
  • trueなら中の処理を実行する
  • falseになったら繰り返しを終える

break

繰り返しをその場で終了します。たとえば「探している値が見つかったら、もう続けなくてよい」という場面で使います。

break;
  • 一番内側の繰り返しを抜ける
  • breakより後の処理は実行しない
  • 終了後は繰り返しの次へ進む

continue

繰り返し自体は続けたまま、今の1回分だけ残りの処理を飛ばします。「条件に合わない回だけスキップする」と考えるとわかりやすいです。

continue;
  • 今の回だけを途中で終える
  • 次の回の判定へ進む
  • 除外したい値があるときに便利
