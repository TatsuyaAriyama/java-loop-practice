Boolean Lesson

真偽値と条件式を理解する

条件式は、最後に必ずtrueかfalseへ変わります。if文へ進む前に、この判定の目を鍛えます。

int score = 82;
boolean passed = score >= 60;

System.out.println(passed); // true
System.out.println(!passed); // false
boolean

trueかfalseだけを入れる型

boolean は真偽値の型です。入れられる値はtrueかfalseだけです。

condition

条件式は判定結果を返す

score >= 60 のような式は、計算結果としてtrueかfalseになります。ここを読めないと分岐は読めません。

comparison

比較演算子で値を比べる

==、!=、>、<= などで値を比べます。代入の=とは別物です。

!

否定は結果を反転する

!passed はpassedの反対です。trueならfalse、falseならtrue。短いですが、読み飛ばすと危険です。

&&

両方trueならtrue

&& は「かつ」です。条件を狭くします。片方でもfalseなら、全体はfalseです。

||

どちらかtrueならtrue

|| は「または」です。条件を広げます。片方でもtrueなら、全体はtrueです。
